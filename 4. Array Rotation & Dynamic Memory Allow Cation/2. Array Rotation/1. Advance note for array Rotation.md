### ðŸ”¹ Idea:

Rotate the array one step left **`k` times**.
Each left rotation: move first element to the end.

### ðŸ”¹ Time Complexity:

* Each rotation = O(n)
* k rotations = O(n Ã— k)

### ðŸ”¹ Code:

```cpp
#include <iostream>
using namespace std;

void rotateLeftByOne(int arr[], int n) {
    int temp = arr[0];
    for (int i = 0; i < n - 1; i++) {
        arr[i] = arr[i + 1];
    }
    arr[n - 1] = temp;
}

void bruteForceRotate(int arr[], int n, int k) {
    for (int i = 0; i < k; i++) {
        rotateLeftByOne(arr, n);
    }
}
```

## âœ… **Stage 2: Better Solution (Using Temp Array)**

### ðŸ”¹ Idea:

1. Store the first `k` elements in a temp array.
2. Shift the rest.
3. Copy back the `k` elements at the end.

### ðŸ”¹ Time Complexity: O(n)

### ðŸ”¹ Space Complexity: O(k)

### ðŸ”¹ Code:

```cpp
void betterRotate(int arr[], int n, int k) {
    k = k % n;
    int temp[k];

    for (int i = 0; i < k; i++)
        temp[i] = arr[i];

    for (int i = k; i < n; i++)
        arr[i - k] = arr[i];

    for (int i = 0; i < k; i++)
        arr[n - k + i] = temp[i];
}
```

## âœ… **Stage 3: Optimal Solution (Reversal Algorithm)**

### ðŸ”¹ Idea:

1. Reverse the first `k` elements.
2. Reverse the remaining `n - k` elements.
3. Reverse the entire array.

### ðŸ”¹ Time Complexity: O(n)

### ðŸ”¹ Space Complexity: O(1) (in-place)

### ðŸ”¹ Example:

```cpp
Input:  [1, 2, 3, 4, 5], k = 2  
Step 1: Reverse first 2 â†’ [2, 1, 3, 4, 5]  
Step 2: Reverse last 3 â†’ [2, 1, 5, 4, 3]

```

### ðŸ”¹ Code:

```cpp
void reverse(int arr[], int start, int end) {
    while (start < end) {
        swap(arr[start], arr[end]);
        start++;
        end--;
    }
}

void optimalRotate(int arr[], int n, int k) {
    k = k % n;
    reverse(arr, 0, k - 1);        // Step 1
    reverse(arr, k, n - 1);        // Step 2
    reverse(arr, 0, n - 1);        // Step 3
}
```
