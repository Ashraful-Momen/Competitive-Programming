## ðŸ“Œ `vector<vector<int>> matrix;`

* This declares a **2D vector** in C++.
* Think of it like a **dynamic matrix** (rows Ã— columns).
* Each element of `matrix` is itself a `vector<int>`.

So:

```cpp
vector<vector<int>> matrix;
```

means:
ðŸ‘‰ `matrix` is a vector (rows)
ðŸ‘‰ Each row is itself a vector of integers (columns)

---

## ðŸ”¹ Example 1: Empty 2D vector

```cpp
vector<vector<int>> matrix;
```

Right now, `matrix` has **0 rows**. You must push values.

```cpp
matrix.push_back({1, 2, 3});  // row 0 = [1,2,3]
matrix.push_back({4, 5, 6});  // row 1 = [4,5,6]
```

Now:

```
matrix = {
  {1,2,3},
  {4,5,6}
}
```

## ðŸ”¹ Example 3: Input for Competitive Programming
